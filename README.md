# Advitiya Bharat - Indian Cultural Heritage Platform

A comprehensive, interactive website celebrating India's rich cultural heritage through art, craft, and tradition. Built with modern web technologies and featuring beautiful animations, responsive design, and seamless user experience.

## 🌟 Features

### **Navigation & Layout**
- **Unified Navigation Bar**: Consistent navigation across all pages with "Advitiya Bharat" branding
- **Mobile-Responsive Design**: Hamburger menu for mobile devices with smooth animations
- **Glassmorphism Effects**: Modern glass-like UI elements with backdrop blur
- **Animated Backgrounds**: Floating particles and gradient backgrounds throughout

### **Interactive Elements**
- **Ripple Effects**: Click animations on buttons and interactive elements
- **Scroll-Triggered Animations**: Elements fade in as users scroll
- **Hover Effects**: Smooth transitions and glow effects on hover
- **Loading States**: Animated spinners and progress indicators
- **Smooth Scrolling**: Seamless navigation between sections

### **Color Palette**
- **Primary Colors**: Dark Navy (#09122C), Dark Burgundy (#2D1B3D)
- **Accent Colors**: Vibrant Red (#E67E22), Light Orange (#D35400)
- **Consistent Theming**: Applied across all pages for unified experience
